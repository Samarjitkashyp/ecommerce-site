# Laravel E-commerce Website

## Features
- ✅ Amazon style header with dropdowns
- ✅ Category slider with custom navigation
- ✅ Modern product cards with add to cart
- ✅ Flipkart style footer
- ✅ Owl Carousel sliders
- ✅ Bootstrap 5
- ✅ Font Awesome icons
- ✅ Responsive design

## Requirements
- PHP >= 8.0
- Composer
- MySQL
- Node.js & NPM (optional)

## Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/myproject.git

## Install dependencies
- composer install
- npm install

## Environment setup
- cp .env.example .env
- php artisan key:generate

## Database configuration
- Create database in MySQL
- Update .env file with database credentials

## Run migrations
- php artisan migrate
- Start the server


- php artisan serve
